Bilancio Familiare - Vercel ready

Istruzioni rapide per il deploy su Vercel (senza usare terminale):
1. Crea un repository su GitHub (es. 'bilancio-familiare').
2. Estrai i file dello ZIP e carica TUTTI i file nella root del repo GitHub (usando 'Add file' -> 'Upload files').
3. Vai su Vercel (https://vercel.com/new), scegli 'Import from Git' e collega GitHub, poi seleziona il repo creato.
4. Vercel rileverà automaticamente Vite/React e farà la build. Se necessario, imposta Build Command: `npm run build` e Output directory: `dist`.
5. Dopo il deploy, copia l'URL prodotto e incollalo in una pagina WordPress con un iframe.

Se vuoi, posso guidarti passo-passo durante il caricamento su GitHub e il deploy su Vercel.
